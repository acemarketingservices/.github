# Ace Marketing Miami

Creator-management agency in Miami, currently working with **15 active OnlyFans creators**.

We handle the operational side of being a creator — content production, posting cadence, fan engagement, and the financial back-office (invoicing, commissions, payouts) — so creators can focus on the work they want to make.

Website: [acegirls.ai](https://acegirls.ai)

## What's in this org

This GitHub org hosts the internal tooling that keeps Ace running day-to-day. Most repos are private; the public ones live here for tooling or external integrations.

| Repo | Purpose |
|---|---|
| **`ace-ops`** | Team-facing automations: content pipeline, social-media posting, and operational workflows. Runs on a shared Mac Studio. |
| **`ace-exec-ops`** | Owner-only financial tooling: Mercury banking, QuickBooks sync, OnlyFans payout reconciliation, weekly commission invoicing, monthly P&L. |
